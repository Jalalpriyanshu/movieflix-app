# Movie Booking App

A modern React app for browsing and booking movies, inspired by BookMyShow.

## Features

- Browse movies
- View movie details
- Book tickets (mock)
- Login/Signup (mock)
- FAQ and Contact pages
- Responsive design

## Setup

1. Clone the repo or copy the files.
2. Run `npm install`.
3. Run `npm start` to launch the app.
